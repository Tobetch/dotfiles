## How to use?

First of all, execute "git clone" command as folloings!

```
$ mkdir ~/git
$ cd ~/git
$ git clone https://github.com/mc-chinju/dotfiles
```

Next, execute shell script!

```
$ ~/git/dotfiles/setup.sh
```

That's all!
